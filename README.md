#Pre requisitos para rodar o codigo:
Versão do Python: Python 3.14.2

Um projeto simples e interativo em Python onde o jogador precisa adivinhar um número secreto gerado aleatoriamente pelo computador.

📌 Sobre o projeto

Neste jogo, o computador escolhe um número aleatório entre 0 e 100, e o jogador deve tentar adivinhar qual é esse número.

A cada tentativa, o programa dá dicas:

🔼 Se o número digitado for menor que o secreto
🔽 Se for maior
🎉 Até o jogador acertar!
🚀 Tecnologias utilizadas
Python 🐍
Biblioteca padrão:
random
🧠 Conceitos aplicados

Este projeto utiliza conceitos fundamentais de programação:

while → repetição até acertar
if / elif / else → tomada de decisões
input() → interação com o usuário
int() → conversão de dados
import random → geração de números aleatórios
▶️ Como executar
Certifique-se de ter o Python instalado
Copie o código para um arquivo, por exemplo:
jogo_adivinhacao.py
Execute no terminal:
python jogo_adivinhacao.py
🎮 Como jogar
Digite um número entre 0 e 100
Leia a dica fornecida pelo programa
Continue tentando até acertar o número secreto
